Code Offsets

Code Offsets are a novelty gift whose proceeds go to various non-profits related to IT education. The concept is a play on the idea of carbon offsets. Developers know they're going to occasionally write bad code, so you buy Offsets in denominations of code lines to pay it forward. I created the responsive site, developed a shopping cart, wrote the copy, and designed the offsets for print.
